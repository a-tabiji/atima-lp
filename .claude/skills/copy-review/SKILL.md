# コピーレビュー

LPのコピーライティングを評価し、改善案を提案してください。

## 引数
- `$ARGUMENTS`: 対象セクション名（省略時はLP全体）

## ブランドトーン&マナー
- **トーン**: 温かみ、親近感、安心感
- **ターゲット**: 子育て中のファミリー（特にママ）
- **避けるべき表現**: 堅すぎる、営業的、テクニカル
- **推奨する表現**: 共感的、具体的、行動を促す

## 分析項目

### 1. キャッチコピー（h1, h2）
- 「体験に集中しながら、プロが写真撮影」— 価値提案が明確か？
- 代替案の提案（3案以上）
- 感情に訴える力、具体性、差別化

### 2. サブコピー（p, hero-subtitle）
- キャッチコピーを補完しているか
- 具体的なベネフィットが含まれているか
- 長すぎないか（モバイルでの可読性）

### 3. セクションタイトル
- 「こんな悩みはありませんか？」「At Imaの3つの特徴」等
- より魅力的・具体的な表現への改善案

### 4. CTA文言
- 「モニターに応募する」「カメラマンに応募する」「提携の相談をする」
- 行動のハードルを下げる文言への改善案
- マイクロコピー（CTA下のテキスト）の効果

### 5. ユーザーの声
- 感想テキストの自然さ・説得力
- 共感ポイントの明確さ
- 具体的なエピソードの有無

### 6. FAQ
- 質問文が想定ユーザーの言葉になっているか
- 回答が安心感を与えるトーンか

## 出力形式

セクションごとに:
```
### [セクション名]
**現状**: [現在のコピー]
**評価**: [良い点・改善点]
**改善案A**: [案]
**改善案B**: [案]
**推奨**: [A/B どちらか + 理由]
```
