# UX 総合監査チーム

3名の専門家チームで LP の UX を多角的に監査してください。

## チーム構成

以下の3名のチームメイトを並列で起動する:

### Teammate 1: UXアナリスト
**担当**: LP全体の導線・構造分析
**指示内容**:
```
あなたはUXアナリストです。At Ima（あっといま）のLPを分析してください。

担当タスク:
1. index.html を読み、セクション構成を把握する
2. ユーザー導線を分析（認知→興味→理解→信頼→行動）
3. CTA の配置・数・文言を評価
4. 認知負荷の分析（テキスト量、選択肢の数、視覚的休息）
5. モバイルUXの課題を特定

出力形式:
- 各項目を5段階で評価
- 問題点は「なぜ問題か」の根拠付き
- 改善案は優先度（高/中/低）付き
- 最後にクイックウィン3つを厳選
```

### Teammate 2: ペルソナテスター
**担当**: 3ペルソナ視点でのLP体験シミュレーション
**指示内容**:
```
あなたはユーザーリサーチャーです。3つのペルソナになりきってLPを評価してください。

ペルソナ:
A. 32歳子育てママ（メインターゲット）— 2歳児あり、共働き、Instagram利用
B. 22歳写真専門学校生（カメラマン候補）— ポートフォリオ充実させたい
C. 45歳公園管理者（施設提供者）— 来場者増がKPI、リスク回避的

各ペルソナについて:
1. index.html を上から下まで読み進め、各セクションでの感情・反応を記録
2. 「離脱しそうになるポイント」を特定
3. CTA到達時の心理状態（応募する気になったか？残る不安は？）
4. 10段階の利用意向スコアと理由

他のチームメイトの発見と比較し、共通する課題があれば強調してください。
```

### Teammate 3: コピー＆コンバージョンアナリスト
**担当**: コピーライティングとCVR改善分析
**指示内容**:
```
あなたはコピーライター兼CROスペシャリストです。LPのコピーとコンバージョン導線を分析してください。

担当タスク:
1. index.html の全テキストを抽出し、トーン&マナーの一貫性を評価
2. キャッチコピー（h1）の訴求力を評価し、代替案を3つ提案
3. CTA文言の改善案（行動ハードルを下げる表現）
4. 説得構造（AIDCA）の評価
5. 心理的障壁の分析（価格、品質、プライバシー、手間）
6. 各障壁に対するLP上の解消要素の有無

他のチームメイトが指摘した離脱ポイントに対するコピー改善案も提案してください。
```

## チーム起動

上記3名のチームメイトを Agent Team として起動する。
各メンバーは `index.html` と `css/style-amber.css` を読み取り専用で分析する。
